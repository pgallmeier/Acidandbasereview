Binary Acids — Formula to Name
HCl, HBr, HF, HI, H₂S — no oxygen present
▼

A binary acid contains H + one nonmetal and no oxygen. Always written (aq) to show it is dissolved in water.

hydro+ [root of nonmetal]+ ic+ acid
FormulaNonmetal / RootNameMemory hint

Binary Acids — Name to Formula
Reverse the process: strip the name, find the element
▼

If the name starts with “hydro-” it is a binary acid. Reverse the three pieces to find the element, then write the formula.

Example: hydrobromic acid
1. Remove “hydro-” →bromic acid
2. Remove “-ic acid” →brom → bromine → Br
3. Write H + symbol →HBr ✓
Watch out: H₂S = hydrosulfuric acid. Sulfur has charge 2−, so you need two hydrogens. Always check the anion charge.

Oxyacids — Formula to Name
Contains oxygen: HNO₃, H₂SO₄, HClO₄, etc.
▼

If the formula contains oxygen, it is an oxyacid. The polyatomic ion ending tells you the acid name ending.

Anion ends in -ATE
→ Acid ends in -IC ACID
sulfate → sulfuric acid
Anion ends in -ITE
→ Acid ends in -OUS ACID
sulfite → sulfurous acid
Special Prefixes — Keep Them!
per-…ate → per-…ic acid   |   HClO₄ = perchloric acid
hypo-…ite → hypo-…ous acid  |   HClO = hypochlorous acid
FormulaAnionName
HNO₃nitrate (-ate)nitric acid
HNO₂nitrite (-ite)nitrous acid
H₂SO₄sulfate (-ate)sulfuric acid
H₂SO₃sulfite (-ite)sulfurous acid
H₃PO₄phosphate (-ate)phosphoric acid
HClO₄perchlorate (per-+-ate)perchloric acid
HClOhypochlorite (hypo-+-ite)hypochlorous acid

Oxyacids — Name to Formula
Read the ending, find the anion, balance with H+
▼

No “hydro-” means oxyacid. The name ending tells you the anion. Then add enough H+ to balance the charge.

Name ends in -IC ACID
→ anion is -ATE
Find -ate ion, add H+ equal to its charge
Name ends in -OUS ACID
→ anion is -ITE
Find -ite ion, add H+ equal to its charge
Ion Charge Reference — Tells You How Many H+ to Add
NO₃⁻ / NO₂⁻ → 1− → 1 H
SO₄²⁻ / SO₃²⁻ → 2− → 2 H
PO₄³⁻ → 3− → 3 H
ClO₄⁻, ClO₃⁻, ClO₂⁻, ClO⁻ → 1− → 1 H

Bases — Formula to Name
NaOH, Fe(OH)₃, NH₃ — fixed-charge and transition metals
▼

Most bases are ionic hydroxides. The key question: is it a fixed-charge metal (Groups 1, 2, Al, Zn) or a transition metal that needs a Roman numeral?

Fixed-Charge (Groups 1, 2, Al, Zn)
metal name + hydroxide
No Roman numeral needed. Count OH⁻ to confirm.
NaOH → sodium hydroxide
Ca(OH)₂ → calcium hydroxide
Transition Metal (Fe, Cu, Cr, Co...)
metal + (Roman numeral) + hydroxide
Count the OH⁻ groups = metal charge = Roman numeral.
Fe(OH)₂ → iron(II) hydroxide
Fe(OH)₃ → iron(III) hydroxide
FormulaTypeName
NaOHGroup 1, fixed 1+sodium hydroxide
Ca(OH)₂Group 2, fixed 2+calcium hydroxide
Al(OH)₃fixed 3+aluminum hydroxide
Fe(OH)₂transition, 2 OH⁻iron(II) hydroxide
Fe(OH)₃transition, 3 OH⁻iron(III) hydroxide
Cu(OH)₂transition, 2 OH⁻copper(II) hydroxide
NH₃special caseammonia

Bases — Name to Formula
Read the Roman numeral, find the charge, write OH⁻ groups
▼

Check for a Roman numeral. If present, that is the metal charge. Add that many OH⁻ groups. If no Roman numeral, use the fixed charge for that metal.

Fixed Charge Reference
Na+, K+, Li+ (Group 1) → 1 OH⁻
Ca2+, Mg2+, Ba2+ (Group 2) → 2 OH⁻
Al3+ (always 3+) → 3 OH⁻
Zn2+ (always 2+) → 2 OH⁻
